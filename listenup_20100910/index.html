Contents of SDK

This software development kit contains:
 
Folder
Contents
listenup/codebase compiled JAR files
listenup/docs documentation
listenup/images GIFs and JPEGs for skins, etc
listenup/php_test test installation for servers running PHP
listenup/uploads folder where files are uploaded
listenup_jsp docs and scripts for Java Server Pages (JSP)

More information on ListenUp can be found on the JavaSonics web site.
