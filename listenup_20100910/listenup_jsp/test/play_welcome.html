Play a Voice Message using ListenUp

Return to Home Test Page.

You should hear a voice message play automatically when the Applet appears.

Troubleshooting:

  • If you just see a blank rectangle, then you may be missing the JAR files. Look in your Java console for details.
  • If you get a popup error dialog with a path to a "listenup.lic" file then the license file is not installed or in the wrong folder. Please move it to the folder listed in the pop dialog.
  • If you see an Applet but hear no sound, then test your computer speakers and try again.
