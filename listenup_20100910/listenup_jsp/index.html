The document will guide you through the installation and testing of ListenUp using JSP (Java Server Pages) .  It assumes you have Tomcat installed and running on your web server.  For more information about Tomcat, visit http://jakarta.apache.org/tomcat/

This document sometimes refers to <tomcatdir>, which shall be understood to mean the installation directory of Tomcat.  For example <tomcatdir>/webapps on your server may expand to /home/tomcat/jakarta-tomcat-4.1.27/webapps
